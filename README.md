✨ Features

Interactive Player Carousel: Browse Arsenal stars like Saka, Ødegaard, and Saliba with smooth slides showing stats and short bios.

Multi-Page Views: Navigate through:

Home: Hero section, latest news, and fan testimonials.

All Posts: Search and filter posts on transfers, matches, and club updates.

Add Post: Contribute articles with title, category, image, and content.

Favorites: Save posts for later reading.

History & Legends: Explore Arsenal’s rich history from Herbert Chapman to the Invincibles.

Fan Zone: Enter giveaways for jerseys, tickets, or signed balls.

About: Learn about the blog’s mission and meet the team.

Light/Dark Theme: Toggle between light and dark modes for comfortable reading.

Giveaway Modules: Interactive forms with trivia questions for fans.

Search Functionality: Quickly find posts by title or category.

Full CRUD Operations: Create, read, update, and delete posts using browser local storage.

🛠️ Built With

HTML5 – Structure and semantic markup.

CSS3 – Styling, animations, responsive design, and dark/light themes.

JavaScript (ES6+) – Interactivity, DOM manipulation, and local storage.

Font Awesome – Icons throughout the interface.
